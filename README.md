# Ed-Donner-model-01---Chat-Model

## 🚀 How to Deploy This Workflow In Your n8n Instance

If you want to test or customize this Agentic AI architecture, you can import the JSON schema directly into your workspace in under 60 seconds.

### 📋 Step-by-Step Installation

1. **Access the File:** Click on the `.json` file listed in this repository directory above.
2. **Download the RAW File:** Click the **Download raw file** button (the download arrow icon located in the top-right corner of the code interface) to save the workflow file to your computer.
3. **Import to n8n:** Open a clean canvas in your n8n workspace. Click the menu icon (three dots) in the top-right corner, select **Import from File**, and choose the downloaded JSON file.
4. **Configure Credentials:** Open the LLM and Vector Store nodes to attach your own environment API keys (e.g., OpenAI, Pinecone, Qdrant). 
5. **Run:** Click **Execute Workflow** to run the pipeline!

---

> ⚠️ **Note on Security:** As per standard open-source protocols, all private API keys and environment variables have been stripped from this public JSON schema. You will need to input your own active credentials inside your local n8n instance to trigger the execution.
